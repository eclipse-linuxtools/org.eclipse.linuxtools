Devhelp Documentation

Library Hover Devhelp supplies hover help for library API documentation available in Devhelp format. It also makes Devhelp documentation browsable and searchable in the Eclipse help system.

Library Hover Devhelp works with the documentation generators used most commonly by the GNOME project libraries, for example GtkDoc. Other kinds of libraries can be supported as well, as long as the API documentation is available in HTML and a *.devhelp2 index file is present.

The documentation must be installed locally, so an internet connection is not needed to use Library Hover Devhelp.
