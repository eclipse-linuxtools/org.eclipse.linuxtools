Overview

The Gcov plugin allows to visualize in eclipse gcov's output (aka *.gcda/*.gcno files).

For more details on gcov, visit the GNU GCov documentation at http://gcc.gnu.org/onlinedocs/gcc-4.7.0/gcc/Gcov.html .
